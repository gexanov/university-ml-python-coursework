# OpenCV Computer Vision Coursework

Public coursework subset with OpenCV scripts, Jupyter notebooks, and assignment documents.

## Scope

- Image preprocessing and geometric transforms.
- Edge detection and feature-processing exercises.
- Small OpenCV scripts for facade/route and logo-image experiments.
- Assignment documents from the course modules.

## Cleanup

This directory intentionally excludes videos, ZIP archives, notebook checkpoints, generated caches, and personal/source images. The published copy keeps the code and notebooks needed to review the coursework without exposing local media files.
