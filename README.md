# University ML and Python Coursework

Coursework archive for Python, machine learning, data analysis, frontend practice, and study presentations.

## Contents

| Directory | Contents |
| --- | --- |
| `ml-course-homeworks/` | Jupyter notebooks and compact datasets from ML homeworks. |
| `data-fusion-track-2/` | Feature engineering, neural network, LightGBM, PyBoost, blending, and stacking scripts. |
| `sentiment-analysis-practice/` | Sentiment-analysis practice work, reports, experiment scripts, metrics, and selected plots. |
| `forecasting-notebook/` | Forecasting notebook and compact training CSV files. |
| `frontend-practice-it-purple/` | Static HTML pages from frontend practice. |
| `english-presentation/` | Study presentation files. |
| `simple-solution/` | Standalone Python solution script and metadata. |

## Cleanup

This public copy excludes large raw datasets, trained model binaries, local dependency folders, notebook checkpoints, temporary page renders, logs, and downloaded reference articles. The kept files are enough to show the work, reproduce the structure, and review the code without publishing local machine state.
